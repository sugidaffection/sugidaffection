## 🧠 My dev philosophy

- ✨ **Zero‑cost abstractions** – speed without paying for what you don't use  
- 🧪 **Type safety first** – let the compiler catch bugs before runtime  
- 🔄 **Async‑first design** – non‑blocking, efficient, concurrent  
- 📦 **Modular** – reusable, testable, maintainable  
- 📖 **Docs as a first‑class citizen** – code that explains itself
